# Проект geometric_lib

## Общее описание

Несколько файлов, предназначенных для расчетов связанных с геометрическими фигурами

## circle.py
_Файл, с функциями для круга_

### area(r)
**принимает радиус круга, возвращает его площадь**
```python
k = area(3)
print(k) # 28.27 ...
```



### perimeter(r)
***принимает радиус круга, возвращает его периметр***
```python
k = perimeter(3)
print(k) # 18.84 ...
```

## square.py
Файл, с функциями для квадрата

### area(r)
принимает сторону квадрата, возвращает его площадь
```python
k = area(3)
print(k) # 9
```

### perimeter(r)
принимает сторону квадрата, возвращает его периметр
```python
k = perimeter(3)
print(k) # 12
```
